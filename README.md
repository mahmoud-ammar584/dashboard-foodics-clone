## Project Overview

The dashboard is built with React.js and provides a clean and modern interface to manage a restaurant business. It includes:

- **Sidebar Navigation**
- **Top Navbar**
- **Dashboard metrics cards**
- **Tables for Inventory, Orders, Products, and Reports**
- Fully responsive design using **CSS Grid and Flexbox**
- Dummy data populated for demonstration purposes

---

## Features

- **Home Dashboard:** Shows metrics like Total Orders, Revenue, Customers, and Low Stock Items.
- **Inventory Management:** Displays current inventory items with ID, stock, and unit.
- **Orders Management:** View all orders with status, customer, and total.
- **Products Management:** Lists all products with price and category.
- **Reports:** Displays business reports like Monthly Sales, Top Product, and New Customers.

---

## Technologies Used

- **React.js** - Front-end library
- **React Router v6** - Routing between pages
- **CSS** - Custom styling for the dashboard
- **JavaScript** - Front-end logic and data handling
